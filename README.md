<br>
WDC 可以將無法用taskkill或指定PID來關閉的視窗，進行關閉
<br>
**AutoHotkey_2.0.18_setup.exe** 為 **必要安裝檔案**
<br>
<br>
首先安裝**AutoHotkey_2.0.18_setup.exe**，安裝完即可刪除<br>
WDC.exe是程式本體，點開即用，無需進行安裝
<br>
<br>**配置說明**<br>
1. 關閉視窗數量：1~10<br>
2. 視窗名稱：依照工作管理員顯示名稱輸入<br>
3. 設定執行按鍵：F1~F12<br>
4. 開啟後自動執行：預設未勾選<br>
5. 3秒後自動關閉：預設未勾選<br>
